# Task_2
Создать программу обработки текста учебника по программированию с использованием классов: Символ, Слово, Предложение, Знак препинания и др. (состав и иерархию классов продумать самостоятельно). 
1.	Заменять табуляции и последовательности пробелов одним пробелом.
2.	Вывести все предложения заданного текста в порядке возрастания количества слов в каждом из них.
3.	Во всех вопросительных предложениях текста найти и напечатать без повторений слова заданной длины.
4.	Из текста удалить все слова заданной длины, начинающиеся на согласную букву.
5.	В некотором предложении текста слова заданной длины заменить указанной подстрокой, длина которой может не совпадать с длиной слова.
